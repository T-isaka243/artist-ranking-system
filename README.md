# Artist Ranking System (ARS)

## Version
0.1.0

## Overview

Artist Ranking System (ARS) は、
週間アーチストランキングを管理し、
月間・半期・年間・歴代ランキングを自動集計するシステムです。

## Features

- Weekly Ranking Import
- Song Ranking
- Artist Ranking
- Excel Export
- HTML Export
- Static Website Generator
